# FormFormationAtion

A simple, interactive form builder that lets you visually construct form schemas and export them as JSON. Built with React — and with a lot of help from [Claude AI](https://claude.ai) by Anthropic.

## Getting Started

**1. Clone the repository**
```bash
git clone https://github.com/rifflere/FormFormationAtion
```

**2. Navigate into the project**
```bash
cd FormFormationAtion
```

**3. Start the dev server**
```bash
npm run dev
```

Then open the local URL shown in your terminal (usually `http://localhost:5173`) and you're good to go.
